## 🎯 How to Play

1. Watch the color sequence Simon shows you
2. Repeat the sequence by clicking the colors in order
3. Each round adds one more step — how far can you go?

## ✨ Features

- 🔊 Sound effects for each color
- 📈 Progressive difficulty — sequence grows each round
- 🏆 High score tracking
- 📱 Mobile-friendly touch support

## 🚀 Play Locally

```bash
git clone https://github.com/aasimansari1/Simon-Says-Game.git
# Open index.html in your browser
```
